Quilter Advisor Assistant A RAG-based question answering system built on Quilter advisor support documentation. 

Setup

Clone the repository Install dependencies by running Cell 1 Add your OpenAI API key to the .env file

Running the Notebook If you have the source PDFs, place them in a folder named quilter pdfs - advisor support material in the project root and run all cells from the beginning. If you do not have the source PDFs, a pre-built ChromaDB index is included. Start from Cell 6 and run all cells from there. I will email you an openAI key along with the submission link.
